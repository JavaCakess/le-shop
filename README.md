# le-shop #

## What's le-shop? ##

le-shop is a virtual shopping program written
in C that uses the curses library. It is intended to
be humurous.

### Controls ###

Key           | Action
------------- | -------------
Up            | Move selection up
Down          | Move selection down
B             | Buy selected item
Q             | Quit program
N             | Navigation Menu
